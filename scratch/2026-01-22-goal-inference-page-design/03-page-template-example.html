Online Bayesian Goal Inference

Understanding human intentions through observed actions

Yuer Tang, Tao Gao

UCLA · Computational Vision and Learning Lab

Paper Code Demo

How do we understand what others are trying to achieve? This project implements a real-time Bayesian framework that infers human goals from observed actions. As a person navigates toward unknown destinations, the system continuously updates its beliefs about their intentions—demonstrating how AI can reason about human behavior even when that behavior is imperfect.

How It Works

The framework combines three key ideas from cognitive science and AI: modeling goals as planning problems, computing how likely actions are under each goal, and using Bayes' rule to update beliefs as new evidence arrives.

1

Model Each Goal as an MDP

For each candidate goal, we compute the optimal value function using Value Iteration. This tells us what a rational agent should do at every position to reach that goal.

V(s) = maxa Σ T(s,a,s') × [R(s,a,s') + γV(s')]
2

Compute Action Likelihoods with Bounded Rationality

Real humans don't always act optimally. We model this using a softmax policy with a temperature parameter—actions that are nearly optimal are still likely, but suboptimal actions become increasingly unlikely.

P(a | s, goal) ∝ exp(Q(s, a, goal) / τ)
3

Update Beliefs with Bayes' Rule

As each action is observed, we update our belief distribution over goals. Actions that are likely under a goal increase our belief in that goal; unlikely actions decrease it.

P(goal | actions) ∝ P(actions | goal) × P(goal)
